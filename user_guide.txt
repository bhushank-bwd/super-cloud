For backend 
1) npm init -y
2) Install the dependencies you need, such as mongodb, express, cors, and dotenv, mongoose@6.10.0, cors,express-validator,bcryptjs, jsonwebtoken

3) for use Context API

    1) create context/Sample.js
        a) export createContext => SampleContext
        b) export write function SampleProvider 
            i) using useState create State 
            ii) return SampleContext.Provider with value => state to provider with item and setItem 

    2) in index.js 
        a) wrap js with SampleProvider
    
    3) in component
        a) import useContext, SampleContext
        b) use it as variable like => const sampleState = useContext(SampleContext);
        c) access SampleContext.Provider state/value using sampleState
